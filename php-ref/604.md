# PHP sleep() 函数



## 定义和用法

sleep() 函数延迟代码执行若干秒。

### 语法

```
sleep(seconds)
```

| 参数 | 描述 |
| --- | --- |
| seconds | 必需。以秒计的暂停时间。 |

### 返回值

若成功，返回 0，否则返回 false。

### 错误／异常

如果指定的描述 seconds 是负数，该函数将生成一个 E_WARNING。

## 例子

```
<?php
echo date('h:i:s') . "<br />";

//暂停 10 秒
sleep(10);

//重新开始
echo date('h:i:s');

?>
```

输出：

```
12:00:08
12:00:18
```
