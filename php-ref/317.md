# PHP hypot() 函数



## 定义和用法

hypot() 函数计算一直角三角形的斜边长度。

### 语法

```
hypot(x,y)
```

| 参数 | 描述 |
| --- | --- |
| x | 必需。边 x 的长度。 |
| y | 必需。边 y 的长度。 |

### 说明

hypot() 函数将会跟据直角三角形的两直解边长度 x 和 y 计算其斜边的长度。或者是从标点 (x , y) 到原点的距离。该函数的算法等同于 sqrt(x*x + y*y)。

## 例子

```
<?php
echo hypot(2,3);
echo hypot(3,6);
echo hypot(3,6);
echo hypot(1,3);
?>
```

输出：

```
3.60555127546
6.7082039325
6.7082039325
3.16227766017
```
