# PHP fmod() 函数



## 定义和用法

fmod() 函数返回除法的浮点数余数。

### 语法

```
fmod(x,y)
```

| 参数 | 描述 |
| --- | --- |
| x | 必需。一个数。 |
| y | 必需。一个数。 |

### 说明

返回被除数（_x_）除以除数（_y_）所得的浮点数余数。余数（r）的定义是：x = i * y + r，其中 _i_ 是整数。如果 _y_ 是非零值，则 _r_ 和 _x_ 的符号相同并且其数量值小于 _y_ 。

## 例子

在本例中，我们将使用 fmod() 函数来返回 5/2 的余数：

```
<?php
$r = fmod(5,2);
echo $r
?>
```

输出：

```
1
```
