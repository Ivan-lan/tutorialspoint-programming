# PHP sin() 函数



## 定义和用法

sin() 函数返回一个数的正弦。

### 语法

```
sin(x)
```

| 参数 | 描述 |
| --- | --- |
| x | 必需。一个数。 |

### 说明

sin() 返回参数 _x_ 的正弦值。参数 _x_ 的单位为弧度。

## 提示和注释

注释：sin() 函数返回的数值介于 -1 和 1 之间。

## 例子

在本例中，我们将计算不同值的正弦：

```
<?php
echo(sin(3));
echo(sin(-3));
echo(sin(0));
echo(sin(M_PI));
echo(sin(M_PI_2))
?>
```

输出：

```
0.14112000806
-0.14112000806
0
1.22460635382E-016
1
```
