# PHP sinh() 函数



## 定义和用法

sinh() 函数返回一个数的双曲正弦。

### 语法

```
sinh(x)
```

| 参数 | 描述 |
| --- | --- |
| x | 必需。一个数字。 |

### 说明

返回 _x_ 的双曲正弦值，定义为 (exp(arg) - exp(-arg))/2。

## 例子

在本例中，我们将返回不同数的双曲正弦：

```
<?php
echo(sinh(3));
echo(sinh(-3));
echo(sinh(0));
echo(sinh(M_PI));
echo(sinh(M_PI_2));
?>
```

输出：

```
10.0178749274
-10.0178749274
0
11.5487393573
2.30129890231
```
