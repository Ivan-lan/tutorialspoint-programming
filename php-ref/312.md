# PHP exp() 函数



## 定义和用法

exp() 函数计算 e 的指数。

### 语法

```
exp(x)
```

| 参数 | 描述 |
| --- | --- |
| x | 必需。一个数。 |

### 说明

返回 e 的 _x_ 次方值。。

## 提示和注释

提示：用 'e' 作为自然对数的底 2.718282。

## 例子

在本例中，我们将对不同的数应用 exp() 函数：

```
<?php
echo(exp(1));
echo(exp(-1));
echo(exp(5));
echo(exp(10))
?>
```

输出：

```
2.718281828459045
0.36787944117144233
148.4131591025766
22026.465794806718
```
