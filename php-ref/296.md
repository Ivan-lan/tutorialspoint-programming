# PHP abs() 函数



## 定义和用法

abs() 函数返回一个数的绝对值。

### 语法

```
abs(x)
```

| 参数 | 描述 |
| --- | --- |
| x | 必需。一个数。 |

### 说明

返回参数 _x_ 的绝对值。如果参数 _x_ 是 float，则返回的类型也是 float，否则返回 integer（因为 float 通常比 integer 有更大的取值范围）。

## 例子

```
<?php
echo(abs(6.7));
echo(abs(-3));
echo(abs(3));
?>
```

输出：

```
6.7
3
3
```
