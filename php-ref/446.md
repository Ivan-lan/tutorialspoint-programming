# PHP SimpleXML 函数




## PHP SimpleXML 简介

SimpleXML 函数允许您把 XML 转换为对象。

通过普通的属性选择器或数组迭代器，可以处理这个对象，就像处理任何其他对象一样。

其中的一些函数需要最新的 PHP 版本。

## 安装

SimpleXML 函数是 PHP 核心的组成部分。无需安装即可使用这些函数。
