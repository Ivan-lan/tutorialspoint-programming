# PHP usleep() 函数



## 定义和用法

usleep() 函数延迟代码执行若干微秒。

### 语法

```
usleep(microseconds)
```

| 参数 | 描述 |
| --- | --- |
| microseconds | 必需。以微秒计的暂停时间。 |

### 返回值

无返回值。

## 提示和注释

注释：在 PHP 5 之前，该函数无法工作于 Windows 系统上。

注释：一微秒等于百万分之一秒。

## 例子

```
<?php
echo date('h:i:s') . "<br />";

//延迟 10 描述
usleep(10000000);

//再次开始
echo date('h:i:s');
?>
```

输出：

```
09:23:14
09:23:24
```
