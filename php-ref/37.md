# PHP array_sum() 函数



## 定义和用法

array_sum() 函数返回数组中所有值的总和。

如果所有值多是整数，则返回一个整数值。如果其中有一个或多个值是浮点数，则返回浮点数。

PHP 4.2.1 之前的版本修改了传入的数组本身，将其中的字符串值转换成数值（大多数情况下都转换成了零，根据具体制而定）。

### 语法

```
array_sum(array)
```

| 参数 | 描述 |
| --- | --- |
| array | 必需。规定输入的数组。 |

## 例子

```
<?php
$a=array(0=>"5",1=>"15",2=>"25");
echo array_sum($a);
?>
```

输出：

```
45
```
